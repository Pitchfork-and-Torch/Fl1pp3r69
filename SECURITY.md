# Security

## Scope

Fl1pp3r69 is a **manifest-driven Flipper Zero field-ops suite** for authorized research on hardware you own or are explicitly permitted to test. It organizes captures under CASEFILE operations with SHA-256 manifests and optional desktop exfil.

## What this project does not ship

- Network exploitation frameworks
- Region TX bypass / jamming packs
- Ungated bruteforce or exploit binaries
- Cloud telemetry or phone-home code

## Safe use

1. Use only on **owned** tags, radios, and systems (or with written authorization).
2. Keep OPSEC defaults on; treat captures as sensitive.
3. Prefer FAP-only installs unless you understand full firmware builds.
4. Verify release artifacts and builds from this GitHub organization only.

## Reporting

Open a [GitHub issue](https://github.com/Pitchfork-and-Torch/Fl1pp3r69/issues) for non-sensitive bugs. For sensitive findings, use GitHub private vulnerability reporting if enabled on the repository.

## Support the work

Fl1pp3r69 is free and open source. Feature requests welcome via Issues.
